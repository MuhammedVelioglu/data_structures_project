## Project Description
-"This repository contains a menu-based C program that demonstrates fundamental data structures and algorithms."

-"Thanks to the menu-based structure, each of the 10 given tasks can be executed independently within a single program, and the correctness of the algorithms is demonstrated through the obtained outputs."
-“Menü tabanlı yapı sayesinde, verilen 10 isteğin her biri tek bir program içerisinde bağımsız olarak çalıştırılabilmekte ve elde edilen çıktılar üzerinden algoritmaların doğru şekilde uygulandığı gösterilmektedir.”

## Included Topics
1. Sparse Matrix Representation  
2. Tower of Hanoi  
3. Infix to Postfix (Shunting Yard Algorithm)  
4. Max Heap and Min Heap  
5. Doubly Linked List  
6. Circular Linked List  
7. Array Insertion and Deletion  
8. Graph Cluster Detection  
9. DFS and BFS Traversal  
10. Tree Creation Using Array  

## How to Compile and Run
- Open the `.c` file using Dev-C++ or any C compiler  
- Compile and run the program  
- Select operations from the menu
